INSTALAÇÃO DO TYPEBOT LOCALHOST NO MESMO SERVIDOR DO WHATICKET
===============================================================

PASSO 1: Executar como root os comandos abaixo:

apt update && sudo apt upgrade

apt-add-repository universe

apt install python2-minimal

apt install docker-compose

usermod -aG docker ${USER}

PASSO 2: Copiar para o servidor o arquivo docker-compose.yml

PASSO 3: Executar o comando abaixo para instalar o typebot:

docker-compose up -d
